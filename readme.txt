== Description ==
With this plugin you will be able to set an expiration date for certain categories assigned to posts and products. This is specially useful if you want to create a \"featured\" or \"new arrivals\" section in your WooCommerce cart or in your blog. 

== Installation ==
1.   Upload the folder gf-edit-entries to the /wp-content/plugins/ directory
2.   Activate the plugin through the ‘Plugins’ menu in WordPress
3.   That’s it!
